Scratch Project Page

Name Project

Description: I put a background of a person doing gymnastics. Then I had my name come in as different letters and they moved around the screen. The letters also changed colors as they moved around the screen. I also had a forever loop running a drum going back and forth across the bottom of the screen. Our rubic stated that we had to have 3 unique behaviours for each letter and that goal was accomplished in this project.

  • How to add a background
  • How to add letters
  • How to use a forever loop
  • How to move sprites(letters)
  • How to change colors of sprites(letters)
  • We used blocks from the event, style, etc.

      Final Project (Game)

      Description: In this project we had to create a game where we had a start page, a question page and an actual game. In this project I decided to create a maze with multiple levels. You would guide your sprite throughout the maze using the arrow keys. If he hit one of the walls then he would bounce off the walls, not allowing the person to cheat. I also had a timer so if the person playing took too long then the game would end.

      • How to make a sprite not touch walls
      • How to draw in a background
      • How to add a timer and when the timer runs out switch to another background
      • How to add a question page so when someone click on a certain sprite it will switch to that page.
